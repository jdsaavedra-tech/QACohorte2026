# 🚀 Cypress Automation Repository - XAcademy

## 📌 Descripción

Repositorio destinado al almacenamiento y desarrollo de automatizaciones realizadas con Cypress como parte del curso de QA Automation de XAcademy.

El objetivo de este proyecto es centralizar prácticas, ejercicios, casos de prueba automatizados y ejemplos de implementación utilizando Cypress para pruebas End-to-End (E2E) sobre aplicaciones web.

---

## 🛠️ Tecnologías utilizadas

- JavaScript
- Cypress
- Node.js
- npm

---

## 🎯 Objetivos del repositorio

- ✅ Practicar automatización de pruebas E2E
- ✅ Implementar buenas prácticas de testing
- ✅ Aprender manejo de:
  - Selectores
  - Fixtures
  - Hooks
  - Assertions
  - Custom Commands
  - Requests API
  - Manejo de waits
  - Reportes
- ✅ Mantener ejemplos reutilizables para futuros proyectos

---

## 📂 Estructura del proyecto

```bash
cypress/
│
├── e2e/                # Casos de prueba
├── fixtures/           # Datos mock y archivos estáticos
├── support/            # Commands y configuraciones
├── downloads/          # Archivos descargados durante pruebas
├── screenshots/        # Evidencias de fallos
└── videos/             # Grabaciones de ejecución
```

## ⚙️ Instalación

### 📥 Clonar el repositorio

```bash
git clone <URL_DEL_REPOSITORIO>
```

### 📁 Ingresar al proyecto

```bash
cd nombre-del-repo
```

### 📦 Instalar dependencias

```bash
npm install
```

---

## ▶️ Ejecutar Cypress

### 🖥️ Abrir Cypress en modo interfaz

```bash
npx cypress open
```

### ⚡ Ejecutar pruebas en modo headless

```bash
npx cypress run
```
